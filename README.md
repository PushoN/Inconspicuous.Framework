Inconspicuous.Framework
=======================

Inconspicuous.Framework is an MVVM framework for Unity3D/C#. It has the following dependencies:

* DryIoc (https://bitbucket.org/dadhi/dryioc)
* UniRx (https://github.com/neuecc/UniRx)
